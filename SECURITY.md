# Security Policy

If you discover a potential security issue, please do not publicly disclose it.

Please report it by email:

- `yangkem3@gmail.com`

Include:

- What is affected
- Reproduction steps
- Potential impact
- Recommended priority

For this repository, security issues generally refer to:

- unintended data exfiltration in packaged examples
- hard-coded credentials or unsafe local configuration
- unsafe execution hooks in runtime scripts

Until a patch is available, avoid spreading the issue publicly.
