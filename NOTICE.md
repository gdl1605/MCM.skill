# Notice

mcm.skill is an open-source skill package for Chinese mathematical modeling contests.

This repository contains distilled workflows, reusable modeling rules, output contracts,
writing templates, lightweight code snippets, and package metadata. It is designed to help
an AI coding agent split contest problems, choose contest-safe modeling routes, prepare
data inspection and validation plans, generate baseline implementation scaffolds, and turn
results into Chinese contest-paper material.

This repository does not redistribute contest statements, official attachments, datasets,
answer workbooks, full papers, or large verbatim excerpts from any source material. Evidence
labels such as `2021-C066` or `2025-C132` are used only as compact references to summarized
patterns observed during distillation.

The included modeling and writing rules are heuristics, not official contest guidance and
not a guarantee of award-winning results. Users are responsible for checking contest rules,
academic integrity requirements, data licenses, and the correctness of generated models,
tables, figures, and written conclusions before submission.

If you contribute new examples, rules, templates, or snippets, please avoid adding private
data, copyrighted source documents, full contest attachments, credentials, or
case-specific files that are not reusable outside one project.
